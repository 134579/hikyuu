Welcome

Hikyuu trys to investigate the securities and futures markets for systematic trading policies research. It mostly serves the stimulation and invaluation of trading policis,testing the performance metrics of all kinds of systematic trading policies and the exploration of the best one.Previously it was architected to use console commands like Linux shell scripts for interacting between inputs and outputs,which highly eased the research work.In the present C++ and Python are combined in.Probably a GUI operating interface will be added in some days,but GUI is not the main purpose because GUI maight not benefit more for the research work comparing the GUI and console commands.

“貔貅交易研究室”是研究试验证券及期货市场系统化交易策略的开源平台。主要用于交易策略的模拟和评估，检验各种系统化交易策略的性能，及最佳化交易策略的探索与研究。早期开发主要采用命令行交互方式，以便于探索和研究。目前的实现主要是采用C++/Python的结合方式。后期不排除提供GUI操作界面的可能，但提供GUI不是主要目的，因为GUI和命令行相比，界面相对固定不利于探索研究。
